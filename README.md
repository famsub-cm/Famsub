# Famsub – Share Subscriptions, Simplified

**Famsub** is a platform designed to make sharing digital subscriptions easy and secure. Users can either create a subscription offer or join an existing one, helping everyone save money and enjoy premium services.

## 🚀 Key Features
- User authentication (sign up, login)
- Create or join subscription offers
- Simple chat to coordinate with other users
- Personal dashboard to manage your subscriptions

## 🛠 Tech Stack
- **Frontend**: Nuxt.js 3
- **Backend**: NestJS
- **Database**: PostgreSQL
